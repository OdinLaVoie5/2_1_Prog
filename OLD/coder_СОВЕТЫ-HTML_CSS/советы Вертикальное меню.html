Вертикальное меню для сайта

Вертикальное меню делается на основе списка, чаще маркированного. По умолчанию все элементы списка <li>...</li> располагаются вертикально, занимая по ширине всю ширину элемента контейнера <ul>...</ul>, который в свою очередь занимает всю ширину его блока-контейнера.

Вертикальное меню помимо элементов списка <li>...</li> может содержать заголовки, иконки, картинки. С его помощью можно оформлять комментарии на сайте, список категорий и т.д.

P.S. Урок будет дополнен другими примерами по мере возможности.

1. Вертикальное меню с заголовком

Простой элегантный вариант оформления. Подходит для стилизации категорий на сайте. При наведении элемент списка меняет цвет ссылки.

2. Вертикальное меню в стиле «схема метро»

3. Элегантное вертикальное меню с градиентом

Градиент и разделяющая нижняя граница добавляют глубину элементам списка. Для списка задан полупрозрачный белый фон, через который просвечивается основной фон родительского контейнера. В качестве фона можно устанавливать как цвет, так и градиент или фоновое изображение.
