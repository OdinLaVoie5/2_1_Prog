2.3. CSS позиционирование


CSS позиционирование управляет местоположением элементов на странице. Изменять положение элементов на странице можно с помощью css-свойств position и float.

1. Позиционирование элементов на странице с помощью свойства position

Свойство position определяет порядок, в соответствии с которым выбранный элемент отображается на веб-странице. По умолчанию все элементы располагаются последовательно один за другим в том порядке, в котором они определены в html-документе ({position: static}). Свойство не наследуется.

Блочный элемент (p, div, h1 и др.) занимает 100% ширины родительского элемента (по умолчанию — body). Поэтому блочные элементы отображаются один под другим в соответствии с разметкой страницы.

Строчный элемент (em, strong, span и др.) занимает ширину, которая соответствует ширине содержимого внутри него. Поэтому строчные элементы отображаются рядом друг с другом.

Свойство position вместе со значениями top, right, bottom и left отображает элемент с нарушением обычного порядка, смещая его на заданное расстояние. При позиционировании элементов можно использовать как положительные, так и отрицательные значения.

position
Значение:
static Значение по умолчанию. Элементы отображаются последовательно один за другим в том порядке, в котором они определены в html-документе.
relative Позволяет смещать элемент относительно его текущего положения в структуре html-документа вверх, вправо, вниз или влево (зависит от того, какое задано значение: top / right / bottom / left). Остальные элементы размещены на странице относительно его первоначального положения, без смещения.
absolute Можно задавать только для блочных (display: block) и строчно-блочных (display: inline-block) элементов.
Позиция элемента смещается относительно родительского элемента вверх, вправо, вниз или влево (зависит от того, какое задано значение: top / right / bottom / left), при этом нарушается порядок отображения элементов.
Элементы, следующие за ним, могут попасть под него.
На положение также влияет значение свойства position родительского элемента.
Так, если у родителя значение position установлено как static или родителя нет, то отсчет положения ведется от верхнего края окна браузера.
Если у родителя значение position задано как relative / fixed / absolute, то отсчет положения элемента ведется от края родительского элемента.
Можно вкладывать один блок с position: absolute в другой блок с position: absolute.
fixed Позволяет зафиксировать элемент в нужном месте страницы. При прокрутке страницы положение элемента на странице не меняется. Элемент можно сместить вверх, вправо, вниз или влево (в зависимости от заданного значения: top / right / bottom / left).

1.1. Свойство bottom

Свойство устанавливает смещение между нижним внешним краем поля позиционированного элемента и нижним краем блока-контейнера. Применяется к элементам, для которых свойство position не равно static. Не наследуется.

bottom
Значение:
auto Значение по умолчанию. Вычисляемое значение свойства равно нулю.
длина Смещение задается в единицах длины.
% Процентные значения вычисляются относительно высоты блока-контейнера.
inherit Наследует значение от родительского элемента.

1.2. Свойство left

Свойство устанавливает смещение между левым внешним краем поля позиционированного элемента и левым краем блока-контейнера. Применяется к элементам, для которых свойство position не равно static. Не наследуется.

left
Значение:
auto Значение по умолчанию. Для элементов с position: relative значение свойства left всегда эквивалентно значению свойства right.
длина Смещение задается в единицах длины.
% Процентные значения вычисляются относительно высоты блока-контейнера.
inherit Наследует значение от родительского элемента.

1.3. Свойство right

Свойство устанавливает смещение между внешним краем правого поля позиционированного элемента и правым краем его блока-контейнера. Применяется к элементам, для которых свойство position не равно static. Не наследуется.

right
Значение:
auto Значение по умолчанию. Для элементов с position: relative значение свойства left всегда эквивалентно значению свойства right.
длина Смещение задается в единицах длины.
% Процентные значения вычисляются относительно высоты блока-контейнера.
inherit Наследует значение от родительского элемента.

1.4. Свойство top

Свойство устанавливает смещение между внешним краем верхнего поля позиционированного элемента и верхним краем его блока-контейнера. Применяется к элементам, для которых свойство position не равно static. Не наследуется.

top
Значение:
auto Значение по умолчанию. Для элементов с position: relative значения свойств top и bottom, равные auto, будут вычисляться в ноль.
длина Смещение задается в единицах длины.
% Процентные значения вычисляются относительно высоты блока-контейнера.
inherit Наследует значение от родительского элемента.

2. Обтекание элементов на странице с помощью свойства float

В обычном порядке блочные элементы отображаются, начиная с верхнего края окна браузера по направлению к нижнему краю.

Свойство float позволяет перемещать любой элемент в нужное место, выравнивая его по левому или правому краю веб-страницы или содержащего его элемента-контейнера. При этом остальные блочные элементы будут его игнорировать, а строчные элементы будут смещаться вправо или влево, освобождая для него пространство и обтекая его.

Плавающий блок принимает размеры своего содержимого с учетом внутренних отступов и рамок блока. Верхние и нижние отступы margin плавающих элементов не схлопываются.

Свойство не наследуется.

Свойство float применяется как к блочным элементам, так и к строчным элементам. В качестве значений указывается одно из трех: float: left / right / none.

При использовании свойства float для блочных элементов обязательно задавать ширину. Тем самым браузер создаст место для другого содержимого. Но если совокупная ширина всех столбцов окажется меньше доступного места, то последний элемент спустится вниз.

При этом вертикальные отступы (margin) обтекаемых элементов не схлопываются с отступами соседних элементов, в отличие от обычных блочных элементов.

Чтобы предотвратить смещение следующего элемента, для отмены правил обтекания применяют свойство clear и одно из его значений: left / right / both.

Значение float: none; отменяет любое перемещение, применяется только в отношении элемента, для которого уже установлено обтекание. Например, для элемента задан стилевой класс со значением float: left;. Вы хотите отменить это обтекание на определенной странице вашего сайта, для чего вы задаете этому элементу специфичный селектор с float: none;.

Правило {overflow: hidden;} предотвращает отображение фона или границ под плавающими элементами.

2.1. Отмена обтекания элементов с помощью свойства clear

Свойство определяет, как будет располагаться элемент, идущий следом за плавающим элементом. Свойство отменяет обтекание с одной или обоих сторон элемента, установленное свойством float. Принимает следующие значения: clear: left;, clear: right;, clear: both;, clear: none; и clear: inherit;.

clear
Значение:
left Отменяет обтекание по левой стороне, смещая элемент вниз. Правое обтекание остается в силе.
right Отменяет обтекание по правой стороне, смещая элемент вниз. Левое обтекание остается в силе.
both Отменяет обтекание с обеих сторон элемента, смещая его вниз.
none Отменяет очистку обтекания, установленное для элементов одной группы.
inherit Наследует значение от родительского элемента.

4. Проблемы позиционирования

1. У элемента с position: absolute ширина равна ширине содержимого, т.е. такой блок не занимает всю ширину родительского блока.
2. Если внутри блока с position: absolute расположены элементы, для которых задано обтекание float, то высота этого элемента будет равна высоте самого высокого из этих элементов.
3. На элемент с position: absolute и положительными значениями left, right, top или bottom не оказывают влияние положительные значения свойства margin.
4. Для одного элемента нельзя одновременно устанавливать свойства position: absolute и float с установленными значениями, но можно задавать position: relative и float.
