Обработка пробелов между буквами и словами

1. Расстояние между буквами и словами

1.1. Расстояние между словами

Свойство позволяет устанавливать интервалы между словами. Можно задавать как положительные, так и отрицательные значения, при отрицательном значении слова могут накладываться друг на друга. При установлении определённого значения оно добавляется к стандартному расстоянию между словами. На значение word-spacing оказывает влияние значение свойства text-align (в случае выравнивания текста по ширине). Наследуется.

word-spacing
Значения:
normal Значение по умолчанию, равное нулю.
Синтаксис: p {word-spacing: normal;}
длина px, pt, cm, em Принимает как положительные, так и отрицательные значения, увеличивая или уменьшая расстояние между словами.
Синтаксис: p {word-spacing: 2px;}

1.2. Расстояние между буквами

Свойство устанавливает расстояние между буквами (величину трекинга) и символами. Может принимать положительные и отрицательные значения. Целесообразно применять для повышения выразительности и читаемости заголовков, определений и пр. Наследуется.

letter-spacing
Значения:
normal Значение по умолчанию, равное нулю.
Синтаксис: p {letter-spacing: normal;}
длина Любое значение будет увеличивать или уменьшать расстояние между буквами. Обычно задается в px или em.
Синтаксис: p {letter-spacing: 2px;}

2. Форматирование пробелов

2.1. Обработка пробелов

Свойство позволяет управлять разделителями (пробелами) между словами и строками текста. Не наследуется.

white-space
Значения:
normal Между словами вставляется только по одному пробелу, дополнительные пробелы отбрасываются.
Синтаксис: p {white-space: normal;}
nowrap Запрещает переносы текста в элементе, за исключением применения <br>.
Синтаксис: p {white-space: nowrap;}
pre Пробелы в тексте не игнорируются, браузер отображает дополнительные пробелы и переносы строк.
Синтаксис: p {white-space: pre;}
pre-wrap Сохраняет пробелы в тексте, делая разрывы строк там, где это необходимо.
Синтаксис: p {white-space: pre-wrap;}
pre-line Удаляет лишние пробелы, за исключением случаев <br>.
Синтаксис: p {white-space: pre-line;}

2.2. Настройка табуляции

По умолчанию табуляция делает отступ, равный восьми пробелам. Для изменения величины отступа, получаемого с помощью клавиши ТAB, можно использовать как целое значение, так и задать длину с помощью рх. Значения свойства игнорируются, когда установлено одно из трёх значений pre-line, normal или nowrap свойства white-space.

tab-size
Значения:
целое число Любое целое положительное число.
Синтаксис: p {tab-size: 4;}
длина Значение отступа, указываемое в рх, поддерживается не всеми браузерами.
Синтаксис: p {tab-size: 4px;}
