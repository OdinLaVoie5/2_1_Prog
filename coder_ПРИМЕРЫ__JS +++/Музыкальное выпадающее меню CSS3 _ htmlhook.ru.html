Музыкальное выпадающее меню CSS3

09.07.201221.12.2015 Автор Виктор Клим5

Музыкальное выпадающее меню HTML5 CSS3 имеет эффект анимации CSS3 (отскок) при наведении курсора на элементы меню. Также использованы элементы HTML5 аудио чтобы добавить музыку в меню.

Вот конечный результат:

Демо

Шаг 1. HTML

Сначала для меню подготовим макет HTML. Каждый элемент меню (которое на самом деле представляет собой неупорядоченный список элементов) имеет якоря, или вложенные уровни.

<ul id="nav">
<li><a href="#">Menu element</a>
<ul>
<li><a href="#">Submenu element</a></li>
.....
</ul>
</li>
<li><a href="#">Menu 4</a></li>
.....
</ul>

При наведении курсор мыши на элемент из списка меню, будет анимация масштаба пункта меню.

Шаг 3. HTML5 JavaScript

Теперь пришло время добавить музыку. Для этого подготовим новый пустой массив (чтобы создать наши аудио объекты), а затем, когда страница готова инициализировать музыку:

// variables
var aLoops = []; // sound loops

// initialization
addEvent(window, 'load', function (event) {

    // load music files
    aLoops[0] = new Audio('media/background.ogg');
    aLoops[0].volume = 0.3;
    aLoops[1] = new Audio('media/button.ogg');
    aLoops[1].volume = 1.0;
    aLoops[2] = new Audio('media/button_click.ogg');
    aLoops[2].volume = 1.0;

    aLoops[0].addEventListener('ended', function() { // loop background sound
        this.currentTime = 0;
        this.play();
    }, false);
    aLoops[0].play();
});

Добавляем обработчики различных событий: Наведение курсора мыши, MouseOut и нажатие click:

// all the buttons
  var aBtns = document.querySelectorAll('#nav li');

  // onmouseover event handler
  addEvent(aBtns, 'mouseover', function (event) {
  aLoops[1].currentTime = 0;
  aLoops[1].play(); // play click sound
  });

  // onmouseout event handler
  addEvent(aBtns, 'mouseout', function (event) {
  aLoops[1].currentTime = 0;
  aLoops[1].pause(); // play click sound
  });

  // onclick event handler
  addEvent(aBtns, 'click', function (event) {
  aLoops[2].currentTime = 0;
  aLoops[2].play(); // play click sound
  });

Можно отключить музыку, оставить только звуковое сопровождение меню при наведении курсора.
